A Simple Shift Cipher With Ruby

Apr 27th, 2014 | Comments

Crytography was born thousands years ago, dating back to early day of the Roman which Ceasar used a very simple form of substitution cipher to encrypt military message. Nowaday cryptography is much more complex and virtually not solveable in some cases (well until quantum computing).

Today I am going to show how to implement the Caesar’s Cipher with improvements:

  • Based on Unicode table
  • Accept case sensitive

Read on you’ll be surprised how easy it is with Ruby!

Concept

The concept is pretty much the same with Caesar’s Cipher in which character is shifted by a key to the left to generate encrypted text. You can read more about it on WWW.

Implementation

Unicode characters are mapped to Unicode table and are represented with number in bytes.

English alphabets are basic Latin which are mapped in range 32 -> 127 on Unicode table. See the table
